# Install Project

## Back - API Django rest

Install depedencies
```bash
pip install -r requirements.txt
```

Migrate
```bash
py manage.py migrate
```

Launch server
```bash
py manage.py runserver
```

http://127.0.0.1:8080/api
http://127.0.0.1:8080/docs
http://127.0.0.1:8080/admin
